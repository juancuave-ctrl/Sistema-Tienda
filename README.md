\# 🏪 Sistema-Tienda



Sistema de Gestión de Tienda desarrollado con Node.js, Express y MySQL.



\## 📌 Descripción



Aplicación web para administrar usuarios y productos de una tienda.



El sistema cuenta con:



\- Backend desarrollado con Node.js y Express.

\- Base de datos MySQL.

\- Frontend desarrollado con HTML, CSS y JavaScript.

\- API REST para comunicación entre módulos.
