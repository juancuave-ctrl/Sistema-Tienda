# Sistema de Gestión de Tienda

## Descripción

Aplicación web desarrollada como evidencia del programa de formación del SENA.

El sistema permite administrar usuarios y productos mediante una API REST desarrollada con Node.js y Express, conectada a una base de datos MySQL.

## Tecnologías Utilizadas

- Node.js
- Express
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Estructura del Proyecto

Sistema-Tienda/

├── backend/

├── frontend/

├── database/

└── docs/

## Instalación

### 1. Clonar o descargar el proyecto.

### 2. Abrir la carpeta backend.

### 3. Instalar dependencias.

```bash
npm install
```

### 4. Configurar la base de datos MySQL.

Crear la base de datos llamada:

```
tienda
```

Importar el archivo:

```
database/tienda.sql
```

### 5. Ejecutar el servidor.

```bash
node server.js
```

### 6. Abrir el frontend.

Ejecutar Live Server sobre:

```
frontend/index.html
```

## Funcionalidades

### Usuarios
